# 웹 어드민 요구사항 정리

## 1) 기술 스택
- React 18+ (Vite)
- TypeScript
- UI 라이브러리 자유 선택 (예: shadcn/ui, Ant Design)

## 2) 화면별 요구 기능
1. 대시보드
- 오늘 예산 현황
- 참여자 수
- 지급 포인트

2. 예산 관리
- 일일 예산 설정/조회
- 룰렛 참여 취소(포인트 회수)

3. 상품 관리
- 상품 CRUD (목록, 등록, 수정, 삭제)
- 재고 관리

4. 주문 내역
- 주문 목록 조회
- 주문 상태 변경
- 주문 취소(포인트 환불)

## 3) 필수 구현 항목
- 테이블 목록 UI
- CRUD 폼
- 백엔드 API 연동

## 4) 평가 기준
- 웹 어드민은 UI 디자인보다 기능 동작 여부를 중심으로 평가

## 5) 제출/배포 관련 (웹 어드민 해당)
- 어드민 웹 배포 URL 제출 필수
- 프론트엔드/어드민은 Vercel GitHub 연동 자동 배포 사용 가능
